# Mainnet Go/No-Go Checklist

This checklist is not mainnet approval.

- Testnet deploy evidence required.
- BscScan verification evidence required.
- Multisig confirmation required.
- feeRecipient confirmation required.
- TierUpdater confirmation required.
- DEX factory confirmation required.
- Initial pair confirmation required.
- Owner approval required.
- No mainnet deploy without explicit owner instruction.
